Prompt 1 · Start plain
Fill in your three answers, then copy.
I'm a [role] in [area]. My ideal clients are [who]. Be my referral strategist — find the kinds of businesses whose clients tend to need someone like me.
Prompt 2 · Make it search
Now use web search to find five real ones near me, each with a website or public phone.
Prompt 3 · Add the why + opener
For each, add why their clients need me, when I become relevant, and a warm opener I could send.
Prompt 4 · Make it a tracker
Put it in a simple table I can keep — name, contact, opener, last contacted, and next follow-up two weeks out.
